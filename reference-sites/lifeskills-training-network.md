# Reference Site: Life Skills Training Network

**URL**: https://lifeskills.vercel.app
**Saved**: 2025-12-18
**Why I like it**: Clean, professional design for a service-based organization

## Design Notes

### Overall Aesthetic
- Clean, professional, and approachable
- Balances credibility with accessibility
- Service-focused organization design

### Layout & Structure
- **Navigation**: Simple horizontal menu (Home, About Us, Training, Blog, Contact, Enroll)
- **Hero Section**: Large headline "Master Life Saving Skills" with value proposition
- **Content Organization**: Clear sectional breaks with descriptive headings
- **Footer**: Multi-column layout with quick links, programs, and contact information

### Key Design Features
- **Statistics Display**: Prominent metric cards (2,500+ students, 15+ years, 100% certification rate) to establish trust
- **Service Highlights**: Four-column grid showcasing core offerings
- **Call-to-Action Buttons**: Multiple conversion points ("Start Training," "Learn More," "Schedule Consultation")
- **Contact Integration**: Phone, email, physical address, and calendar scheduling link

### Visual Elements
- Logo present in header and footer
- .webp image format (modern optimization)
- Newsletter subscription form in footer
- Social media links

### Takeaways for Future Projects
- Statistics/metrics build trust quickly
- Multi-column service grids work well for showcasing offerings
- Multiple CTAs throughout the page increase conversion opportunities
- Clear sectional organization improves scannability
