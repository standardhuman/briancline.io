## Design Decisions (from Brainstorming)

These decisions were made through collaborative Q&A:

| Question | Decision |
|----------|----------|
| **Primary audience** | Professional network (entrepreneurs, tech folks, men's work community) |
| **Main impression** | Trust & credibility - "This guy is legit" |
| **Desired action** | Reach out directly (email/form) AND schedule a conversation (Calendly) |
| **Positioning** | Entrepreneur/Builder - lead with businesses and projects |
| **Projects to feature** | All: SailorSkills, Men's Circle, ANT Manufacturing, MCP Tools + sandbox (Boatwise, FireShift, needs-are-normal) |
| **Personal elements** | Integrated throughout - adventures inform the builder |
| **Visual aesthetic** | Clean & minimal (Apple/Stripe vibes) |
| **Tech stack** | Vite + vanilla JS + Tailwind CSS |
| **Background format** | Brief narrative prose (not timeline or grid) |
| **Social links** | LinkedIn, GitHub, Strava |

### Key Tagline

**"Building what works. On land and sea."**

Brian refined this from the initial proposal "Building businesses that work. On water and land."

## What Was Built

### Page Structure

1. **Navigation** - Fixed header with name + "Let's Connect" CTA
2. **Hero** - Full viewport, name, tagline, headshot placeholder, CTA button
3. **Current Focus: SailorSkills** - Service description, tech platform value prop, 1,000+ services stat
4. **Projects** - 6-card grid (Men's Circle, ANT, MCP Tools, Boatwise, FireShift, needs-are-normal)
5. **Background** - 3-paragraph narrative weaving experiences
6. **Connect** - Contact form (Formspree) + Calendly placeholder side-by-side
7. **Footer** - Copyright + social icons (LinkedIn, GitHub, Strava)

## Design Reference

- **Secondary text**: `gray-600` (#4B5563)

- **Section headings**: 3xl-4xl, bold
